# pos
### 项目说明
此项目是将四年前的一个pos项目进行重构，原本使用的是jquery+electron+Vue多页面，我的任务是将它去electron，用vue单页面的形式重构成一个网页版的pos收银系统。

## Project setup
```
npm install
```
